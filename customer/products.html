How to choose

Start with the workload, not the processor name

Choose EDGE LITE whenYou need a compact embedded core, onboard wireless and display or camera integration.
Choose EDGE GATEWAY whenYou need multiple network paths, field-installation options and replaceable radio modules.
Choose customisation whenThe standard platform is close, but form factor, I/O, sensing or cost requires a programme-specific variant.
Add Lifecycle Platform whenYou need a maintained Linux base, remote delivery and a defined long-term update path.
