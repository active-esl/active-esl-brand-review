# Active ESL — public brand review

Public GitHub Pages stakeholder pack for reviewing the proposed Active ESL
brand direction.

This repository contains only the static review site and approved concept
assets. Working collateral, production files and project history remain in a
separate private repository.

## Local preview

```bash
python3 -m http.server 8080
```

Open http://127.0.0.1:8080/

## Published site

https://active-esl.github.io/active-esl-brand-review/

## Content order

1. **Guidelines pages** — overview, foundation, identity, colour/type, imagery, production, review
2. **Original PCB image concepts** — generated assets in `assets/images/`
3. **Figma boards** — PNG exports in `assets/boards/` (auto-shown on `boards.html`)

The identity and imagery are concepts for stakeholder review. They are not
trade-mark-cleared final artwork, and AI-generated product imagery is disclosed
within the site.
