Figma page: https://www.figma.com/design/SkU2A8GkJdSmT82wTLWsxr?node-id=17-2

Export tips:

1. Select each frame in Figma.
2. Export at 2× PNG.
3. Keep width around 1600–3200 px.
4. Commit into this folder; the boards page auto-detects the files.
