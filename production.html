04 / Production

Built for screens, boards and boxes

The identity must survive real manufacturing constraints — not only presentation mockups.

Web

  • SVG first, PNG/WebP fallbacks
  • Symbol from 24 px
  • Gradient: hero accent only

Print / PDF

  • Vector PDF / SVG
  • Use solids
  • Confirm CMYK proof

PCB silkscreen

  • Mono outlined paths
  • Start ≥ 0.25 mm line/gap
  • Confirm fab capability

Enclosure

  • Mono or three filament colours
  • Start ≥ 0.8 mm feature
  • Relief ≥ 0.6 mm

UV / eufyMake

  • Flat substrate or film
  • Texture within machine limit
  • Proof colour and adhesion

Non-negotiable

  • Never stretch or rebuild the mark
  • No gradient on PCB or print-critical work
  • Never brand as “Electric Edge”
  • Verify supplier / machine before release
